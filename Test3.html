Elections Ontario
At my time at Elections Ontario I got to learn more about Ontario's election system. Below is a sample of some of the geospatial products I construced during my time as a intern.
Pilot Project

This is a map I created for a pilot project, with the goal of mapping out all the polling locations within one electoral district. The goal of this project is to eventually have it expanded out to all the electoral district to give more information to voters, to allow them to more easily find the polling districts.

Post Seconadry Insutions by Electoral District

This is a geovisualization of all the post secondary instutions by electoral district within Ontario. This was created in order to help asses stratagies to increase youth voting. The process invovled manually collecting all the post secondary instutions and the there repsective campus and geoporocessing the addreses.

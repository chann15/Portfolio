Hi, I'm Nicholas, a fourth-year student about to graduate from the University of Toronto, majoring in Human Geography and minoring in GIS and History.

Elections Ontario

At my time at Elections Ontario I got to learn more about Ontario's election system. Below is a sample of some of the geospatial products I construced during my time as a intern.

Pilot Project

This is a map I created for a pilot project. This pilot aimed to map out all the polling locations within one electoral district with the eventual goal of expanding it to all electoral districts. This in turn would help inform voters about the location of voting polls within there are electoral district.

Post Secondary Institutions by Electoral District

This is a geovisualization of all the post-secondary institutions by electoral district within Ontario. This was created to help assess strategies to increase youth voting. The process involved manually collecting all the post-secondary institutions and their respective campus and georeferencing the addresses.

GGR376, Geovisualization

In this course, I learned how to increase the efficiency of communicating my findings through various techniques and medians, such as static and dynamic maps, 3D and 2D maps as well as story maps. Below are some of the projects that I worked on during this course.

Geovisualization of the Standard Mortality Ratio against the

COVID-19 Infection Rate within Ontario

This is a geovisualization of the standard mortality ratio (SMR) against the COVID-19 infection rate. The SMR was calculated using covid-19 mortality data, and population size. Similarly, the COVID-19 infection rate was calculated using such data. The data was calculated and cleaned from Statistics Canada.

GGR375, Introduction to Programming in GIS

This course taught me how to use libraries such as Geopanadas and other GIS libraries, to help analyse and visualize spatial data. Below is the final term project that I did for GGR375.

Examining the Relationship between COVID-19 Vaccination Accessibility and the COVID-19 Infection Rates

This project involved examining the relationship between Covid-19 vaccination accessibility and Covid-19 rates. This was done through Python using packages such as geopandas and matplotlib. This project involved examining how accessible the COVID-19 vaccination clinics whereby public transit and then overlaying that information above a choropleth map of normalized covid-19 cases within Toronto and running an autocorrelation analysis.
